# Intro
- OpenClaw is an open-source personal AI assistant that runs locally on your machine (Mac, Windows, or Linux)
- It acts as an autonomous agent that can manage emails, calendars, files, browser automation, and more
- Controllable via chat apps like WhatsApp, Telegram, Discord, Slack, Signal, iMessage, and others
- Supports multiple AI backends: Claude, GPT, or local models
- 50+ integrations including Gmail, GitHub, Spotify, Obsidian
- Homepage: (https://openclaw.ai/)
- GitHub: (https://github.com/openclaw/openclaw)
